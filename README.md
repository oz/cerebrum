# Cerebrum

Cerebrum is a simple, non-optimized, brainfuck interpreter in Go.

Run with:

```shell
$ go get github.com/oz/cerebrum
$ go install github.com/oz/cerebrum
$ echo "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++." | cerebrum
Hello World!
```
